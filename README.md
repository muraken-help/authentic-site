# Authentic / オーセンティック Webサイト

画像デザイン案をもとに作成した静的Webサイト一式です。

## ファイル構成
- `index.html`：トップページ
- `style.css`：デザイン・レスポンシブ対応
- `script.js`：スマホメニュー
- `assets/authentic-logo.png`：ロゴ画像

## 編集ポイント
- 連絡先は仮の内容です。公開前に電話番号・メールアドレスを差し替えてください。
- 代表写真はプレースホルダーです。実写真を使う場合は `profile-photo` 部分を画像に差し替えてください。
- サービス名は、AI活用・人材育成・事業伴走に寄せています。必要に応じて変更できます。
